# #67 Code Scrubber

A hyper-minimalist, zero-friction string escape and unescape tool designed for developers. Part of the **Fridge Combo** series.

## 1 Tool / 1 Action
The **Code Scrubber** eliminates the cognitive load of choosing between formats. Simply paste your string, and instantly get four essential developer formats. No buttons to click, no modes to switch.

## Key Features
- **Auto-Processing**: Results appear instantly as you type or paste.
- **Quad-Output**: Simultaneous generation of:
  - **HTML Escape**: For safe rendering in web documents.
  - **HTML Unescape**: To restore readable code from entities.
  - **JS/JSON Escape**: For safely embedding strings in code.
  - **JS/JSON Unescape**: To decode backslash-escaped strings.
- **Smart UI**: 
  - **Tap-to-Copy**: Click or tap any result box to copy instantly to your clipboard.
  - **Auto-Expand**: The input area grows vertically to match your content.
  - **Sticky Clear**: The "clr" button stays within reach even with long texts.
- **Privacy-First**: All processing happens locally in your browser. No data ever leaves your device.

## Design Philosophy
- **Minimalist Aesthetics**: Background `#fff`, main text `#333`, and a consistent `15px` border-radius.
- **High Performance**: Optimized for low-spec PCs and mobile devices.
- **Mobile Native Feel**: Specifically tailored for iOS Safari with custom input styling and fixed text-fill issues.

## Technical Specifications
- **No Login / No Tracking / No Ads**
- **Browser-Only**: Pure HTML/CSS/JS implementation.
- **Mobile Friendly**: Fully responsive and optimized for touch interaction.

## How to Use
1. Paste your raw text or escaped string into the top box.
2. View the four auto-generated result boxes.
3. Tap the desired box to copy the result.
4. Use the "clr" button to reset the interface.
